# find_duplicates
## Installation
You need to retrieve, obviously:
> find_duplicates.sh

if you want to get a colored display:
> lib_colors.sh

## Usage
> usage: find_duplicates.sh [-h] [-p path]
>    h : show this help
>    p : find all duplicates within the tree path
>        ex: find_duplicates.sh -p /Users/foo
